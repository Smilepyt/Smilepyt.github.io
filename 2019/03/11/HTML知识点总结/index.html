什么是 DOCTYPE

DOCTYPE是一种标准通用标记语言的文档类型声明，目标是告诉标准通用标记语言解析器要使用什么样的文档类型定义(DTD)来解析文档。

<!DOCTYPE> 声明是用来指示web浏览器用哪个HTML版本来解析该页面。

为了能够很好地显示满足标准的页面，又能最大程度兼容不合法的HTML。 浏览器厂商一般会提供两种浏览器模式：

  • 标准模式（standards mode）：浏览器根据标准规约来渲染页面。
  • 混杂模式（quirks mode）：浏览器采用更加宽松的、向后兼容的方式来渲染页面。

HTML5 新标签

常见的H5新标签：

  • <section> 定义文档中的一个章节。
  • <nav> 定义只包含导航链接的章节。
  • <article> 定义可以独立于内容其余部分的完整独立内容块。
  • <aside> 定义和页面内容关联度较低的内容——如果被删除，剩下的内容仍然很合理。
  • <header> 定义页面或章节的头部。
  • <footer> 定义页面或章节的尾部。
  • <main> 定义文档中主要或重要的内容。
  • <time> 代表日期和时间值。
  • <mark> 代表一段需要被高亮的引用文字。
  • <vidio> 代表一段视频 及其视频文件和字幕，并提供了播放视频的用户界面。
  • <audio> 代表一段声音 ，或音频流 。
  • <canvas> 代表位图区域 ，可以通过脚本在它上面实时呈现图形，如图表、游戏绘图等。
  • <svg> 定义一个嵌入式矢量图 。
  • <math> 定义一段数学公式 。

<canvas>和<svg>比较：

<canvas> 基于像素的，有很多图形函数，可以在js中进行绘制；适合用来处理比较复杂的，实时性比较高的，比如游戏。
<svg> 是矢量的，适合用来处理高保真的、静态的图形图像。

块级元素和行内元素

常见块级元素(block element)：div p form table h1-h6 ol ul li blockquote address等
常见行内元素(inline element)：a span img code input lable select strong等
空元素(empty element)：area br col img input link meta等

块级元素（display:block）：

  1. 独占一行，默认宽度为父级宽度的100%；
  2. 可以设置宽度和高度；
  3. 可以设置 margin 、padding；

行内元素（display:inline）：

  1. 不会独占一行，相邻行内元素排在同一行；
  2. 不可以设置宽度和高度，宽度随内容变化；
  3. 仅可以设置 margin左右、padding左右，上下无效；

inline-block属性： 可以通过设置display:inline-block。既具有 block 元素可以设置宽高的特性，同时又具有 inline 元素默认不换行的特性。使用inline-block 会产生元素间的空隙。

HTML语义化

  1. 让正确的标签做正确的事情；
  2. 在没有CSS样式的情况下，页面依旧以文档的形式展示，便于阅读；
  3. 搜索引擎的爬虫依赖于HTML的标签来确定上下文和各个关键字权重，利于SEO；
  4. 让阅读源码的人更好的划分网站结构，利于阅读和维护。
