vue双向数据绑定

发表于 2019-03-12 | 分类于 web前端

原理

vue是一款基于MVVM模式的轻量级的框架，是一款基于数据驱动、组件化思想的框架。

vue使用数据劫持和订阅发布实现双向数据绑定。

原理图：

通过实例化一个vue对象，对其数据属性进行遍历，通过Object.defineProperty() | Proxy 进行数据劫持,并对模板做编译生成指令对象，每个指令对象绑定一个watcher对象，然后数据赋值的时候触发setter,相应的watcher对象对其再次求值，如果值真有变化，就会通知相应的指令，调用指令的update方法，由于指令是对DOM进行封装，这时调用DOM的原生方法对DOM进行更新，实现数据驱动DOM的变化。同时vue还会对DOM进行事件监听，如果DOM发生变化，vue监听到，就会修改相应的data。

版本变化

Vue3.0中，采用 Proxy 替代 Object.defineProperty() 来实现数据响应。

这将会消除Vue2.x中基于 Object.defineProperty() 的实现存在的一些限制。如：

  1. Object.defineProperty 只能监听属性，而 Proxy 能监听整个对象，省去对非对象或数组类型的劫持，也能做到监听。
  2. Object.defineProperty 无法监听数组的变化，使Vue不得不对数组做了额外的hack。
